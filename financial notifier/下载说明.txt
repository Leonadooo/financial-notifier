# 金融资讯定时推送系统 - 下载说明

## 📦 文件说明

这个zip包包含了运行金融资讯定时推送系统所需的所有文件。

## 🚀 快速开始

### 第1步：解压文件
将zip文件解压到你想要的目录，例如：
- Windows: C:\financial-notifier
- Mac/Linux: ~/financial-notifier

### 第2步：安装依赖

**Windows (CMD):**
```cmd
cd C:\financial-notifier
pip install -r requirements.txt
```

**Windows (PowerShell):**
```powershell
cd C:\financial-notifier
pip install -r requirements.txt
```

**Mac/Linux:**
```bash
cd ~/financial-notifier
pip install -r requirements.txt
```

### 第3步：测试运行

```bash
python scripts/daily_notification.py
```

如果看到"✅ 所有推送渠道成功！"，说明系统运行正常！

### 第4步：配置推送渠道

编辑 `config/notification_config.json` 文件，修改为你的邮箱：

```json
{
  "enabled_channels": ["email"],
  "email": {
    "enabled": true,
    "recipients": ["你的邮箱@example.com"]
  }
}
```

### 第5步：推荐使用GitHub Actions（电脑关机也能收到）⭐

按照 `docs/GITHUB_ACTIONS_TUTORIAL.md` 教程操作，10分钟搞定！

## ❓ 常见问题

### Q: 我应该选择哪种部署方式？

**Windows电脑不一直开机** → GitHub Actions（推荐）
**需要最高稳定性** → 云服务器
**快速测试** → 本地部署

### Q: pip install 失败怎么办？

使用国内镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 如何修改推送邮箱？

编辑 config/notification_config.json：
```json
{
  "email": {
    "enabled": true,
    "recipients": ["你的邮箱@example.com"]
  }
}
```

### Q: 邮件推送失败？

1. 检查邮箱是否开启SMTP服务
2. 使用"授权码"而不是"登录密码"
3. 查看详细文档：docs/DEPLOYMENT_GUIDE.md
