# 📧 QQ邮箱SMTP服务开启指南

## 🎯 为什么要开启SMTP服务？

为了使用邮件推送功能，你需要开启邮箱的SMTP（简单邮件传输协议）服务，获取**授权码**。

**重要：** 邮件推送使用的是"授权码"，不是你的登录密码！

---

## 📝 开启步骤（3步搞定）

### 第1步：登录QQ邮箱

1. 打开浏览器，访问：https://mail.qq.com
2. 输入QQ号和密码登录
3. 进入邮箱界面

### 第2步：进入设置

1. 点击页面顶部的 **"设置"**（⚙️图标）
2. 在设置页面，找到并点击 **"账户"** 选项卡

### 第3步：开启SMTP服务

1. 在"账户"页面，向下滚动，找到 **"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"** 部分

2. 找到 **"SMTP服务"**，点击右侧的 **"开启"** 按钮

3. 验证身份：
   - 选择验证方式：**手机短信验证** 或 **密保问题验证**
   - 按照提示完成验证

4. 验证成功后，系统会显示 **"授权码"**

5. **⚠️ 重要：立即复制保存授权码！**
   - 授权码是一串16位的字符，类似：`abcdefghijklmnop`
   - 只会显示一次！一定要复制保存！

---

## 📋 开启后你会看到的信息

### 服务器配置信息

开启后，页面会显示以下配置信息：

```
SMTP服务器：smtp.qq.com
SMTP端口：465 或 587
加密方式：SSL
```

### 重要信息

- **SMTP服务器地址：** `smtp.qq.com`
- **SMTP端口：** `465`（SSL加密）或 `587`（STARTTLS）
- **你的授权码：** `abcdefghijklmnop`（示例，请使用你自己的）

---

## 🔐 什么是授权码？

### 授权码 vs 登录密码

| 类型 | 用途 | 安全性 |
|------|------|--------|
| **登录密码** | 登录QQ邮箱网页 | ⭐⭐⭐ |
| **授权码** | 第三方应用发送邮件 | ⭐⭐⭐⭐⭐ |

### 为什么用授权码？

- ✅ 更安全，避免密码泄露
- ✅ 可以随时更改授权码，不影响邮箱登录
- ✅ 如果授权码泄露，只需重新生成即可

---

## ⚠️ 重要提示

### 1. 授权码只显示一次

授权码在开启SMTP服务后只显示**一次**，如果忘记了：

**如何重新获取授权码？**

1. 登录QQ邮箱
2. 设置 → 账户
3. 找到SMTP服务
4. 点击 **"生成授权码"**
5. 验证身份
6. 复制新的授权码

### 2. 使用授权码，不是登录密码

在配置金融资讯推送系统时：

**❌ 错误：**
```json
{
  "password": "你的QQ密码123"  // 错误！不要用登录密码
}
```

**✅ 正确：**
```json
{
  "password": "abcdefghijklmnop"  // 正确！使用授权码
}
```

### 3. 定期更换授权码

为了安全，建议每3-6个月更换一次授权码。

---

## 🚀 如何使用授权码？

### 方法1：在配置文件中

编辑 `config/notification_config.json`：

```json
{
  "enabled_channels": ["email"],
  "email": {
    "enabled": true,
    "recipients": ["your_email@qq.com"],
    "smtp_server": "smtp.qq.com",
    "smtp_port": 465,
    "account": "your_email@qq.com",
    "auth_code": "abcdefghijklmnop"  // 这里填你的授权码
  }
}
```

### 方法2：环境变量（可选）

如果你使用GitHub Actions或云服务器部署，可以在环境变量中配置：

```
EMAIL_SMTP_SERVER=smtp.qq.com
EMAIL_SMTP_PORT=465
EMAIL_ACCOUNT=your_email@qq.com
EMAIL_AUTH_CODE=abcdefghijklmnop
```

## ❓ 常见问题

### Q1: 找不到"账户"设置？

**A:**
- 登录QQ邮箱后，点击顶部的 **"设置"**（⚙️图标）
- 在左侧菜单选择 **"账户"**

### Q2: 开启失败，提示"需要验证"？

**A:**
- 这是为了安全，需要验证你的身份
- 选择手机短信验证或密保问题验证
- 按照提示完成验证即可

### Q3: 授权码忘记了怎么办？

**A:**
- 登录QQ邮箱
- 设置 → 账户
- 找到SMTP服务
- 点击 **"生成授权码"**
- 重新验证身份
- 获取新的授权码

### Q4: 邮件推送失败，提示认证错误？

**A:** 检查以下几点：
1. ✅ 是否使用了授权码（不是登录密码）
2. ✅ 授权码是否正确（注意大小写）
3. ✅ SMTP服务器地址是否正确（`smtp.qq.com`）
4. ✅ 端口是否正确（`465` 或 `587`）

### Q5: 开启SMTP后能正常收发邮件吗？

**A:**
- ✅ 完全可以正常使用
- ✅ 不影响你的日常邮件收发
- ✅ 只是多了一个发送邮件的渠道

### Q6: 可以关闭SMTP服务吗？

**A:**
- 可以，但不建议
- 关闭后，第三方应用（如金融资讯推送）将无法通过你的邮箱发送邮件
- 如需关闭：设置 → 账户 → 找到SMTP服务 → 点击"关闭"

## 💡 其他邮箱的SMTP配置

### 163网易邮箱

**SMTP服务器：** `smtp.163.com`
**SMTP端口：** `465`（SSL）或 `587`（STARTTLS）

**开启步骤：**
1. 登录163邮箱
2. 设置 → POP3/SMTP/IMAP
3. 开启SMTP服务
4. 获取授权码

### 126网易邮箱

**SMTP服务器：** `smtp.126.com`
**SMTP端口：** `465`（SSL）或 `587`（STARTTLS）

**开启步骤：** 同163邮箱

### Gmail（国外邮箱）

**SMTP服务器：** `smtp.gmail.com`
**SMTP端口：** `587`（STARTTLS）或 `465`（SSL）

**开启步骤：**
1. 登录Gmail
2. 账户 → 安全
3. 开启"两步验证"
4. 生成"应用专用密码"
